## Features

- 📀 **Browse Drake's Albums** - View all of Drake's studio albums with release years and track counts
- 🎤 **Featured Artists** - Discover which artists are featured on each album
- 🎨 **Album Cover Art** - Visual album covers displayed in list and detail views
- 📱 **SwiftUI Interface** - Modern, clean iOS app interface
- 💾 **JSON Data** - Albums and artist data stored in JSON files
